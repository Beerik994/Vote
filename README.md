# OTABootImageExtractor
An Android APP, dump Android `boot.img` from **full** OTA zip.  
一个Android App，从 **OTA完整包** 中提取 `boot.img`，主要用于Magisk修补。  
仅对小米OTA包测试通过，不保证对其它手机品牌OTA包的兼容性。
